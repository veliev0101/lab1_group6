- part3_pikfactor_LR1.m - скрипт: берет 3 импульса, считает пик-фактор, выводит таблицу
- README.md             - краткая инструкция

## 1) Что делается в ЛР1

В ЛР1 сравнивают 3 одиночных импульса (на интервале 0..T) и смотрят, как форма во времени влияет на:
- энергоспектр |S(f)|^2 (часто показывают еще в dB)
- полосу спектра:
  - по уровню (-20/-30/-40/-60 dB)
  - по концентрации энергии (90/95/99%)
- пик-фактор
- выводы по сравнению импульсов

Ключевая идея: резкие границы во времени дают больше высоких частот (хуже внеполосные компоненты).

## 2) Пик-фактор простыми словами

Берется мгновенная мощность:
p = |a|^2

Далее:
moshchnost_pik  = max(p)
moshchnost_sred = mean(p)

Пик-фактор:
pik_factor    = moshchnost_pik / moshchnost_sred
pik_factor_db = 10*log10(pik_factor)

Проверка:
для прямоугольного импульса a=1 получается pik_factor = 1 (0 dB).

## 4) Как запустить

1) Положить calc_pik_factor.m и part3_pikfactor_LR1.m в рабочую папку MATLAB.
2) Запустить код исполнителя 1 (получить 3 импульса)
3) Запустить:
part3_pikfactor_LR1

## 5) Что получается на выходе

В консоль выводится таблица:
tablica_pikfactor: nazvanie, moshchnost_pik, moshchnost_sred, pik_factor, pik_factor_db

Эта же таблица остается в workspace (можно использовать дальше в общем коде/отчете)
